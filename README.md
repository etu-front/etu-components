# Etu Components Lib

> 常用 components 库

## Usage


## 开发
```bash
$ git clone git@github.com:etu-front/etu-components.git
$ cd etu-components
$ npm install
$ npm run dev
# visit http://localhost:3000
```
## 发布
```bash
$ npm run pub
```
